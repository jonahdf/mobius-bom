# Mobius-Bom
Mobius Intership Coding Challenge

View deployed app [here](https://mobius-bom.herokuapp.com/)

# Features:

- Fully featured table (using MaterialTable library) with filtering, sorting, searching, customizable layout, number of rows
- Editable table: When a row is edited, sends a put request and updates the server
- When edited, updates the "updated at" field with current time
- Fields that shouldn't be edited are not editable
- Delete button that deletes rows locally.

# Technologies Used:

- React.JS
- [Material-table](https://material-table.com/#/)
- Fetch API
- MockAPI
- Heroku
